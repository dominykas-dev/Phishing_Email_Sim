# Phishing_Email_Sim

Makes phising emails to test company's employees if they click the link.

1) Fake Password Reset
2) Fake Invoice or Order Confirmation
3) HR/Internal Communication Impersonation
4) Quishing (QR Code Phishing)

Enter the receivers email and select the type.

- Python for back-end
- HTML for email templates
- Simple database to track results
